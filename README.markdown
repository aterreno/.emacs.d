## Emacs installation

To get started with emacs

- `brew update`
- `brew install emacs --cocoa`
- Make a symbolic link within Macos, that seems the only way to get it running from spotlight

## Setup

- `cd ~`
- `git clone git@github.com:aterreno/.emacs.d.git`
